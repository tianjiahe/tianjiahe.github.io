家禾的第一篇博客文章

序言

因找工作需要,我需要尝试建立自己的博客,
在B站参考了 codesheep.cn 推荐的博客框架, https://hexo.io/zh-cn/, 是用nodejs开发的,正好可以学习一下.

第一章 安装我的博客

从安装nodejs开始的一个完整安装过程:

  1. 从nodejs官网下载nodejs, 会同时安装 nodejs和npm,安装过后用 node -v和 npm -v测试一下.

  2. 将npm的镜像文件下载转到国内的镜像,可加快模块的下载速度.
    执行命令,安装淘宝的cnpm管理器.

    npm install -g cnpm –registry=https://registry.npm.taobao.org
    cnpm -v

注意有两个坑:

在执行 npm 命令时,总是安装失败,经过多此尝试,发现我的网络和系统可能都有问题:

. 首先出现的错误是在修改 npm为国内淘宝的镜像源 时安装进度停止,等了很长时间,最后提示了安装失败.
我怀疑网络连接有问题,就断开宿舍路由器的连接,换上了物联网卡做热点上网,让我的笔记本连上热点(估计手机的4G热点也是可以的),
之后就可以顺利执行 npm 命令了.
. 后来执行其他的 npm 命令时,还是有失败, 后来我发现在开始菜单中有node.js控制台,之后用上就OK了.
建议:最好用node.js自己的控制台(在开始菜单中找 Node.js command prompt) 执行npm.

3.安装 hexo 框架,这个就是用 nodejs 写的博客框架

cnpm install -g hexo-cli
hexo -v

4.创建并测试博客框架,执行如下命令:

mkdir blog ‘ 创建博客目录
cd blog ‘ 进入博客目录
hexo init ‘ 生成博客,初始化博客
hexo s ‘ 启动本地博客服务,启动的是nodejs服务器,从控制台返回的信息

看到控制台提示的网址信息 http://localhost:4000 后,打开浏览器.
可以在浏览器里,看一下博客默认创建的 hello world 文章.

第二章 管理我的博客

创建我的第一个博客文章

1.在控制台,注意在博客的根目录执行: hexo n “我的第一篇文章”
会在下面返回文章文件的保存位置. ./source/_posts

2.修改我的文章,在win10中也具有vim,或是其他的文本编译工具,写的是md格式.
3.修改文件之后,序号顺序执行

hexo clean ‘ 清空数据库
hexo g ‘ 是生成新的
hexo s ‘ 重启服务器

之后再打开浏览器,看一下自己写的博客.

第三章 部署到 github 上

  1. 浏览器登录我的github,并新建一个repository
    新的仓库名称 必须是: 我的名称.github.io
    必须写成: tianjiahe.github.io
    以后就可以在浏览器中输入 tianjiahe ,就可以访问了.

  2. 在博客的跟目录中,下载一个git部署插件.

    cnpm install –save hexo-deployer-git

  3. 设置 _config.yml,找到 deploy: 字段后, 修改如下:

    type: git
    repository: https://github.com/tianjiahe/tianjiahe.github.io.git
    branch: master

  4. 之后,在控制台执行: hexo d // deploy 部署到远端
    也可以合并执行 hexo clean && hexo d

    注意:
    在wind10上执行hexo d 上传到git的操作时会出错,提示需要鉴权,提示输入用户名和密码.
    这里必须用到git的控制台了,才能执行git 相关命令
    找到我的博客文件夹,鼠标选中后右键弹出菜单中, 打开 Git Bash here,
    顺序输入如下命令,配置当前git用户的访问参数:

    $ git config --global user.email "xxxxxxxxx@qq.com"
$ git config --global user.name "tianjiahe"

    执行github客户端鉴权后, hexo deploy 命令也必须再 Git GUI here 的控制台执行,否则还是有错. 

    $ hexo d

  5. 最后可以访问github上的自己的博客地址了, 用户名称.仓库名称.

    https://tianjiahe.github.io/ 家禾的博客
